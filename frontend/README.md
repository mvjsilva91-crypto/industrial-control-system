# 🤖 Sistema de Monitoramento Industrial

Interface web moderna para monitoramento operacional, análise de dados e gestão de processos em ambientes industriais.

---

## 🧩 Visão Geral

O **Smart Factory Dashboard** é uma aplicação front-end desenvolvida com foco em performance, organização e experiência do usuário.

O sistema simula um ambiente de controle industrial, permitindo visualizar métricas, acompanhar eventos e gerenciar diferentes áreas operacionais em um único painel.

---

## 🚀 Principais Recursos

* 📊 Painel com indicadores em tempo real (simulado)
* 🏭 Acompanhamento de produção
* 📈 Visualização de dados e relatórios
* 🚨 Gestão de alertas e ocorrências
* 🔧 Controle de manutenção
* 📜 Registro de atividades (logs)
* ⚙️ Personalização básica do sistema
* 🔐 Fluxo de autenticação simples

---

## 🛠️ Stack Tecnológica

* **React.js** — construção da interface
* **JavaScript (ES6+)** — lógica da aplicação
* **CSS3** — estilização e layout
* **Vite** — ambiente de desenvolvimento rápido
* **Lucide React** — ícones modernos

---

## 🧠 Habilidades Demonstradas

* Arquitetura baseada em componentes
* Gerenciamento de estado com React Hooks
* Estruturação de dashboards complexos
* Organização de código escalável
* Persistência local com `localStorage`
* Criação de interfaces responsivas e intuitivas
